# Security Policy

We take the security and integrity of Vim Royale very seriously. If you have found a vulnerability, please report it ASAP so we can quickly fix the issue.

## Reporting a Vulnerability
For vulnerabilities that impact the confidentiality, integrity, and availability of Vim Royale services, please send your disclosure via [email](mailto:sahookumarjitesh@gmail.com) or this [form](https://www.vimroyale.com/report-vulnerability). For non-security related platform bugs, follow the bug submission guidelines. Include as much detail as possible to ensure reproducibility. At a minimum, vulnerability disclosures should include:

- Vulnerability Description
- Impact
- Steps to Reproduce
- Contact Email (optional - if sending via the form)

## Submission Guidelines
Do not engage in activities that might cause a denial of service condition, create significant strains on critical resources, or negatively impact users of the site outside of test accounts.
